1. 基础语法

1.1 分级标题

1.2 文字修饰

文字修饰主要包含三种：斜体、粗体与下划线

  • 斜体：*我是斜体*，效果：我是斜体
  • 粗体：**我是粗体**，效果：我是粗体
  • 下划线：~~我是下划线~~，效果：我是下划线
  • 斜体+粗体：***我是斜体+粗体***，效果：我是斜体+粗体

1.3 列表

列表主要分两种，有序列表与无序列表，我忘记在哪里看到的了，好像有序列表只需要开始的1，后面会自动排序。

  • 无序列表：无序列表可用+、-、*
  • 有序列表：使用数字

1.4 链接

链接分两种：引用图片与文字链接，区别在于！

  • 文字链接：codewithvicky 语法：[我是链接说明](这里放链接)
  • 引用图片：在[]前加!
